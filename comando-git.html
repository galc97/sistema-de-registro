Comandos Git

  • git clone: Este comando sirve para descargar un repositorio de GitHub a tu computadora. Es como hacer una copia del repositorio en tu máquina local.
  • git log: Este comando te muestra el historial de cambios que se han realizado en un repositorio. Puedes ver quién hizo qué cambio, cuándo lo hizo y qué mensaje dejó asociado al cambio.
  • git pull: Este comando sirve para descargar los cambios que se han hecho en el repositorio remoto a tu computadora. Es como actualizar tu copia local con los cambios más recientes.
  • git status: Este comando te muestra el estado actual de tu repositorio local. Te indica qué archivos han sido modificados, cuáles son nuevos y cuáles no están siendo seguidos por Git.
  • git add: Este comando sirve para agregar archivos al área de preparación (staging area) de Git. Es como decir a Git que quieres incluir estos archivos en el próximo commit.
  • git commit: Este comando sirve para guardar los cambios que has hecho en tu repositorio local. Es como tomar una instantánea de tu proyecto en un momento determinado.
  • git push: Este comando sirve para enviar los cambios que has hecho en tu repositorio local al repositorio remoto. Es como subir tus cambios a GitHub para que otros puedan verlos.
